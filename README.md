# ublacklist-reach-plc
uBlacklist ruleset for blocking all Reach Media plc / Trinity Mirror websites from your search results when used with Ublacklist browser extension

How to use
Add the subscription in uBlacklist

Open Chrome/Firefox etc → Extensions → uBlacklist → Options
Scroll to Subscription → Add a subscription
Paste the raw URL 
https://raw.githubusercontent.com/dickydolittle/ublacklist-reach-plc/refs/heads/main/reach-plc.txt
→ Save
